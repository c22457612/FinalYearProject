Trusted Sites

                

                  The primary place to manage sites you explicitly trust.
                

              

            

            

              Trusted sites allow certain protections to be relaxed where you have explicitly chosen to trust the site.
              Trust does not mean a site is universally safe, and you can remove trust at any time.
            

            

              

                What appears here
                Sites appear because you explicitly trusted them, or because VPT observed privacy-related activity for them.
              

              

                What VPT does not do
                VPT does not read your full browser history, so this page is not a complete list of sites you visited.
              

              

                Manual trust always works
                You can trust a domain here even if VPT has never observed an event for it.

Privacy model

                    
Visible while you review the signals.

                  

                

                

                  These detections show browser API behavior that can suggest fingerprinting or probing, but the stored evidence
                  remains metadata-only.
                

                
    
                  
  • Detection is metadata-only: operation/action, coarse context, counts, states, and safe hostnames where available.
    • 
                  
  • No raw canvas output is stored.
    • 
                  
  • No clipboard contents or copied/pasted binary payloads are stored.
    • 
                  
  • No rendered text extraction is stored.
    • 
                  
  • No SDP bodies are stored.
    • 
                  
  • No ICE candidate strings are stored.
    • 
                  
  • No IP addresses are stored.
    •

Current limits

                    
Make the current scope and privacy boundaries explicit.

                  

                

                
    
                  
  • Controls in this view apply only to implemented surfaces: Canvas, Clipboard, Geolocation, and WebRTC.
    • 
                  
  • Current enforcement is top-frame-only in the repo today.
    •

    • 
                  
  • Stored evidence remains metadata-only: no raw canvas output, no clipboard contents, no SDP bodies, no ICE candidate strings, and no IP addresses.
    •
